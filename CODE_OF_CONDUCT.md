# 贡献者公约行为准则

## 我们的承诺

为了营造一个开放和受欢迎的环境，我们作为贡献者和维护者承诺：无论年龄、体型、身体健全与否、民族、性别认同与表达、经验水平、国籍、个人外貌、种族、宗教或性取向，都友善地对待项目社区中的每一位成员。

## 我们的标准

有助于创造积极环境的行为包括但不限于：

* 使用友好和包容性语言
* 尊重不同的观点和经历
* 耐心地接受建设性批评
* 关注对社区最有利的事情
* 友善对待其他社区成员

不能容忍的行为包括但不限于：

* 使用与性有关的言语或是图像，以及不受欢迎的性关注或举动
* 挑衅、侮辱/贬损的评论，以及人身攻击或政治攻击
* 公开或私下的骚扰
* 未经明确授权便发布他人的资料，如地址、电子邮箱等
* 其他可以被合理地认定为不恰当或者违反职业操守的行为

## 我们的责任

项目维护者有责任为「可接受的行为」标准做出诠释，以及对已发生的不被接受的行为采取恰当且公平的纠正措施。

项目维护者有权利及责任去删除、编辑、拒绝与本行为准则不相符的评论、提交、代码、wiki 编辑、问题及其他贡献，以及暂时或永久性地封禁任何他们认为行为不当、威胁、冒犯、有害的贡献者。

## 适用范围

本行为准则适用于所有项目空间，以及当某人代表项目或其社区时的公开场合。代表项目或社区的情形包括但不限于：使用项目的官方电子邮件地址、通过官方社交媒体账号发布消息、在线或线下活动中被指定为代表。项目维护者可能会进一步定义和澄清代表项目的情形。

## 贯彻落实

对于辱骂、骚扰或其他不被接受的行为，可以通过 [项目维护者邮箱] 联系项目团队来举报。所有的投诉都会得到及时和公正的审查和调查。项目团队有义务对事件举报人严格保密。

## 处理方针

项目维护者将依照下列社区处理方针，判断并处理他们认定违反本行为准则的行为：

### 1. 纠正
**社区影响**：使用不恰当的语言或其他在社区中被认定为不符合职业道德或不受欢迎的行为。

**处理意见**：由项目维护者发出非公开的书面警告，明确说明违规行为的性质，并解释举止如何不妥。或将要求公开道歉。

### 2. 警告
**社区影响**：单个或一系列违规行为。

**处理意见**：警告并对连续性行为进行处理。在指定时间内，不得与相关人员互动，包括主动与行为准则执行者互动。这包括避免在社区空间和外部渠道中的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁
**社区影响**：严重违反社区准则，包括持续的不当行为。

**处理意见**：在指定时间内，暂时禁止以任何形式与社区互动或公开与社区交流。在此期间不得与相关人员公开或私下互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁
**社区影响**：行为模式表现出违反社区准则，包括持续的不当行为、骚扰个人或攻击或贬低某个群体的言行。

**处理意见**：永久禁止在社区内以任何形式公开互动。
